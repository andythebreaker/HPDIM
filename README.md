這是一個閩南語注音直接輸入法的網頁版展示，你可以使用注音輸入法在網頁上直接輸入閩南語。

如何使用
在 main.html 中打開網頁，即可使用閩南語注音直接輸入法。使用方法如下：

點擊輸入框，開啟注音輸入法
輸入閩南語注音
選擇正確的詞語，並點擊插入
注意事項
請使用支援HTML5和JavaScript的瀏覽器。在使用過程中，如有發現任何問題，歡迎回報。
